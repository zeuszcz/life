# Character sprite credits

Character spritesheets come from the **Universal LPC Spritesheet** (ULPC /
Liberated Pixel Cup), assembled via the open-source generator:

- Generator + assets: https://github.com/sanderfrenken/Universal-LPC-Spritesheet-Character-Generator
  (online: https://sanderfrenken.github.io/Universal-LPC-Spritesheet-Character-Generator/)
- Project: Liberated Pixel Cup (LPC) — https://lpc.opengameart.org/
- **License: dual-licensed CC-BY-SA 3.0 and GNU GPL 3.0.**

LPC/ULPC art is by many contributors; primary/base authors commonly credited
across body, hair, clothing and shoes include (non-exhaustive):

- Stephen Challener (Redshrike), Johannes Sjölund (wulax), Manuel Riecke (MrBeast)
- Matthew Krohn (makrohn), Marcel van de Steeg (Bertram), Lanea Zimmerman (Sharm)
- bluecarrot16, JaidynReiman, ElizaWy, and the wider LPC community

Full per-asset attribution is maintained in the generator repository
(`spritesheets/**/*.json` credit files) and on OpenGameArt. As CC-BY-SA 3.0 /
GPL 3.0, this project redistributes the artwork under compatible terms.

Each layer PNG here was cropped to the 4-direction walk block (rows 8–11 of the
universal 64×64 sheet) → 832×256 (13×4 frames; row 0=up, 1=left, 2=down, 3=right).
